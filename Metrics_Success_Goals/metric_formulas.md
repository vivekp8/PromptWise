# Metric Definitions & Formulas

**Prompt Reuse Rate**
= (Prompts Used ≥ 2x) / (Total Approved Prompts)

**Fork Rate**
= (Total Forks) / (Total Sessions Started)

**Token Efficiency**
= (Useful Output Tokens) / (Total Input Tokens)

**Active User %**
= (Users with ≥1 session in past 14 days) / (Total Registered Users)
